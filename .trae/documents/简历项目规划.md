# 简历项目规划

## 一、项目概述
根据设计图实现一个三页的专业简历网站，使用 Vue 技术栈，支持 PDF 导出和可配置数据功能。

## 二、设计分析

### 页面结构
1. **第一页**：
   - 个人信息区（头像、姓名、职位）
   - 基本信息栏（年龄、工作年限、微信、邮箱）
   - 专业技能区（前端基础、框架和库、工程化、AI、其它）
   - 语言能力区（英语、日语）
   - 教育背景区
   - 个人优势区
   - 工作经历区

2. **第二页**：
   - 项目经历区（3个项目，每个项目包含：项目名称、时间、标签、项目描述、核心职责、技术栈、项目成果）

3. **第三页**：
   - 个人开源项目区（3个开源项目，每个项目包含：项目名称、GitHub链接、时间、Star数、项目描述、主要功能、技术亮点、个人贡献）

## 三、技术选型

- **框架**：Vue 3 + Vite
- **样式方案**：Tailwind CSS
- **PDF导出**：html2canvas + jsPDF
- **构建工具**：Vite

## 五、实施步骤

### 阶段一：项目初始化
1. 使用 Vite 创建 Vue 3 项目
2. 配置项目基础结构
3. 安装必要依赖（html2canvas、jsPDF）

### 阶段二：数据结构设计
1. 设计简历数据配置文件结构
2. 将设计图中的示例数据填入配置文件

### 阶段三：组件开发
1. 开发第一页组件（个人信息、技能、教育、工作经历等）
2. 开发第二页组件（项目经历）
3. 开发第三页组件（开源项目）
4. 整合为完整的简历页面

### 阶段四：样式实现
1. 实现基础样式框架
2. 逐个实现各模块样式
3. 进行样式二次确认和优化

### 阶段五：功能实现
1. 实现 PDF 导出功能
2. 测试数据配置的灵活性

### 阶段六：测试与优化
1. 整体功能测试
2. 样式细节优化
3. 性能优化

## 六、关键文件说明

- `src/data/resume-data.js`：存放所有简历数据，包括个人信息、技能、教育背景、工作经历、项目经历、开源项目等，方便用户修改
- `src/components/`：三个页面组件，分别对应设计图的三页
- `src/assets/styles/`：样式文件，统一管理样式

## 七、风险与注意事项

1. **样式确认**：在实现样式过程中，会向用户二次确认关键样式设计
2. **PDF导出**：需要确保导出的 PDF 与网页显示效果一致
3. **响应式设计**：虽然用户未明确要求，但会考虑基础的打印适配
